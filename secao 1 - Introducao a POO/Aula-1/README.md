<h1>Introdução a Programação Orientada a Objetos</h1>

<h2>Aula 1</h2>
- Foi feito código simples na classe Main para calcular a área de dois triângulos, e informar quais deles possui maior área.
- Após analisar o código, é possível verificar que estamos repetindo trechos de código para representar uma mesma entidade (triângulo), é nesse momento que vemos a necessidade de abstrair o triângulo como uma classe.
- **Classe**: possui *atributos* (campos/dados) e *métodos* (funções/operações). Outros recursos da classe: construtores, sobrecarga, encapsulamento, herança e polimorfismo.
- **Objetos** são instâncias da classe (new)
